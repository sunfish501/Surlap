1. 정의

  • 운영자: kev208dev. 문의: kev208dev@gmail.com
  • 이용자: 앱을 설치하여 서비스를 이용하는 모든 사람.
  • 계정: 이메일·소셜 로그인을 통해 생성되는 식별자. 클라우드 동기화 기능 사용 시 필요.
  • 게스트: 로그인 없이 기기 내 저장만으로 사용하는 이용자.

2. 서비스 내용

  • 일정·할 일·기록·시간표·생일 등 개인 데이터의 생성, 편집, 조회.
  • 로그인 시 Supabase 기반 클라우드 동기화 및 백업.
  • 학교 NEIS(시간표·학사일정·급식) 및 공개 스포츠 API 연동(선택).
  • 로컬 알림, 위젯, iCal 내보내기, 테마/일정 공유(읽기 전용 코드).

3. 계정 및 보안

  • 이용자는 계정 정보(이메일, 인증 토큰)를 본인이 안전하게 관리할 책임이 있습니다.
  • 운영자는 계정 도용·부정 사용으로 인한 손해에 대해 책임을 지지 않으며, 의심 정황 발견 시 접근을 제한할 수 있습니다.
  • 회원 탈퇴는 앱 내 프로필 → 설정 → 계정 → 회원 탈퇴에서 즉시 가능합니다. 탈퇴 시 클라우드 데이터는 영구 삭제됩니다.

4. 이용자의 의무

  • 법령 및 본 약관, 공공질서·미풍양속에 반하는 행위를 하지 않습니다.
  • 타인의 권리(저작권·초상권·개인정보 등)를 침해하는 콘텐츠를 저장·공유하지 않습니다.
  • 역공학, 자동화 도구를 통한 비정상 접근, 서버 부하 유발 행위, 보안 우회 시도를 하지 않습니다.
  • 공유 코드를 악용하거나 받은 데이터를 무단 재배포하지 않습니다.

5. 콘텐츠 소유권

이용자가 앱에 저장하는 일정·기록 등 모든 콘텐츠의 소유권은 이용자에게 있습니다. 운영자는 서비스 제공(저장·동기화·표시)을 위한 최소한의 처리 권한만 가지며, 콘텐츠를 분석·광고·제3자 제공에 활용하지 않습니다.

6. 제3자 서비스

  • Supabase: 클라우드 데이터베이스 및 인증(행 수준 보안 RLS 적용).
  • Apple / Google: 소셜 로그인 및 앱 배포.
  • 교육부 NEIS 오픈API: 학교 시간표·급식·학사일정 조회.
  • 공개 스포츠 API: 경기 일정 조회.

각 제3자 서비스의 약관 및 정책이 함께 적용될 수 있습니다.

7. 서비스 변경·중단

운영자는 안정성·보안·기능 개선을 위해 서비스의 전부 또는 일부를 사전 고지 후 변경하거나 중단할 수 있습니다. 긴급한 사유(보안 사고, 외부 API 장애 등)가 있는 경우 사후 고지로 갈음할 수 있습니다.

  • 천재지변, 통신 장애, 제3자 API 장애 등 운영자의 합리적 통제 범위를 벗어난 사유로 인한 손해에 대해 책임지지 않습니다.
  • 이용자가 입력한 데이터의 손실에 대비해 정기적인 백업을 권장합니다.
